# Rentify

Rentify is a property listing website. Where user can rent a property or A landlord rent his property. 

## Installation

To install every node module for backend

```bash
cd .\backend\
npm i
```
To install every node module for frontend
```bash
cd .\frontend\
npm i
```

## Run 
Run Backend

```node
cd.\backend
npm run dev
```

Run Frontend

```node
cd.\frontend
npm run dev
```
